## ğŸš€ Usage

### Basic Usage

```python
from ultra_creative_ui_generator import (
    UltraCreativeUIGenerator,
    UIGenerationRequest,
    BusinessNiche
)

# Initialize generator
generator = UltraCreativeUIGenerator(use_context_engine=True)

# Create request
request = UIGenerationRequest(
    business_niche=BusinessNiche.REAL_ESTATE,
    business_name="DreamHome Realty",
    business_description="Premium real estate platform",
    target_audience="Home buyers aged 25-55",
    key_features=["Property search", "Virtual tours", "Agent matching"],
    dark_mode=True,
    include_pwa=True,
    include_analytics=True,
    seo_optimized=True,
    accessibility_level="AAA"
)

# Generate UI
result = generator.generate_ui(request)

# Access generated assets
print(result.html)  # Complete HTML
print(result.css)   # Optimized CSS
print(result.javascript)  # Interactive JS
print(result.performance_metrics)  # Performance data
```
