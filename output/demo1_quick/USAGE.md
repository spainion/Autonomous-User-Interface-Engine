# Usage Guide for QuickStart App

## Components Used

- hero
- button
- navbar
- card

## Customization

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
}
```

### Adding Components
Use Bootstrap classes:
```html
<div class="card">
    <div class="card-body">
        Your content
    </div>
</div>
```

### Animations
Apply animation classes:
```html
<div class="fade-in">Animated content</div>
```
